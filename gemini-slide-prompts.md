# Google Slides Gemini Prompts
## Vibe Coding — Build Real Things Without Writing Code

---

## HOW TO USE THESE PROMPTS

Open Google Slides, click "Help me create a slide" (Gemini icon), then paste the prompt for each slide.
Each prompt is self-contained — no prior context needed.

**Global design system applied in every prompt:**
- Handwriting-style fonts (Caveat or Patrick Hand)
- Warm cream background (#FFF8F0) with coral/orange as the primary accent color
- Informal, playful, workbook-like feel — not corporate
- Plenty of icons, illustrations, and decorative doodles
- Tables, columns, and diagrams wherever content allows
